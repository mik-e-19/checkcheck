Առաջադրանքի անունը՝ sort_list
Ակնկալվող ֆայլեր՝ sort_list.c
Թույլատրված ֆունկցիաներ՝
--------------------------------------------------------------------------------
Գրել հետևյալ ֆունկցիան՝

t_list  *sort_list(t_list* lst, int (*cmp)(int, int));

Այս ֆունկցիան պետք է որպես պարամետր տրված ցանկը դասավորի՝ cmp ֆունկցիայի ցուցիչը
օգտագործելով կիրառվելիք դասավորվածությունը ընտրելու համար, և վերադարձնի ցուցիչ
դասավորված ցանկի առաջին տարրի վրա։

Կրկնօրինակները պետք է մնան։

Մուտքերը միշտ կլինեն հաստատուն։

Պետք է օգտագործել ձեզ տրամադրված list.h ֆայլում նկարագրված t_list տեսակը։
Պետք է ներառել այդ ֆայլը (#include "list.h"), բայց հանձնել պետք չէ։
Մենք մեր սեփականը կօգտագործենք առաջադրանքը կազմարկելու համար։

Որպես cmp փոխանցված ֆունկցիաները միշտ կվերադարձնեն 0-ից տարբեր արժեք, 
եթե a-ն ու b-ն ճիշտ հերթականությամբ են, 0՝ այլ դեպքում։

Օրինակ, որպես cmp գործածված հետևյալ ֆունկցիան ցանկը կդասավորի աճման կարգով։

int ascending(int a, int b)
{
    return (a <= b);
}
